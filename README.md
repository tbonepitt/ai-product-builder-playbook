# The AI Product Builder Playbook

A microlearning course on AI Product Management. Eight modules, ~95 minutes. Built to teach judgment — not vocabulary.

## Opening the course

Double-click `ai-product-builder-course.html` in Finder. Opens directly in your browser, no server or install needed.

## What's inside

| Module | Topic |
|--------|-------|
| 0 | The Pivot — why the PM bar has shifted |
| 1 | Workflow Thinking — automate work, not features |
| 2 | AI System Design — architect the right stack |
| 3 | Rapid Prototyping — build before you describe |
| 4 | Evaluation Systems — measure quality before you scale |
| 5 | Cost + Latency — own the P&L math |
| 6 | Agentic AI — design agents that act reliably |
| 7 | End-to-End Ownership — from zero to shipped |

Currently a single self-contained HTML file (React + Babel via CDN). Next step is converting to a Vite + React project for local dev and Vercel deployment.
